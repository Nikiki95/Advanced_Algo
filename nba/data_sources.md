# NBA Datenquellen - Vergleich

## Aktuell: fixturedownload.com
- ✅ Kostenlos
- ❌ Nur letzte ~20-30 Spiele
- ❌ Keine Live-Updates
- ⚠️ Limitiert für Short/Medium Memory

## Alternative 1: API-Sports (Empfohlen)
- 💰 Kostenlos: 100 Calls/Tag
- 💰 Pro: €15/Monat (unlimited)
- ✅ ALLE NBA Spiele seit 2015
- ✅ Live Scores, Player Stats, Injuries
- ✅ Komplette Historie für echtes Long Memory
- 🔗 https://api-sports.io/documentation/nba/v2

## Alternative 2: NBA Official API
- 💰 Kostenlos für Entwickler
- ✅ Offizielle Daten
- ⚠️ Rate Limits
- 🔗 https://developer.nba.com/

## Alternative 3: Basketball-Reference (Scraping)
- ✅ Kostenlos
- ✅ ALLE historischen Daten
- ⚠️ Muss gescraped werden (nicht offizielle API)
- ⚠️ Kann sich ändern/brechen
